## Ossium3D 

**Ossium3D** es una plataforma desarrollada en Python que permite realizar un flujo completo de trabajo a partir de estudios DICOM, incluyendo:

- Segmentación de estructuras óseas
- Reconstrucción tridimensional (fémur y pelvis)
- Visualización 3D interactiva
- Modelado paramétrico de componentes protésicos femorales y acetabulares
- Exportación de modelos para manufactura y análisis posterior

---

## Cómo ejecutar la aplicación

1. **Descargar la carpeta completa del proyecto `Ossium3D`**  
   Es importante conservar la estructura original de carpetas.
2. Dentro de la carpeta principal, hacer doble clic sobre el archivo: Ossium3D.bat
3. Se abrirá una ventana de terminal que mostrará el mensaje: Iniciando Ossium3D...
4. Una vez finalizado el proceso de carga, **la interfaz gráfica se abrirá automáticamente**, habilitando todas las funcionalidades del sistema.

---

## Uso del dataset

- Utilizar los estudios que se encuentran en las carpetas "datasetM" o "datasetF" dentro de la carpeta completa del proyecto o en el archivo Datasets.md de este repositorio.
  
---
## ! Notas importantes

- No modificar los nombres de los archivos ni la estructura de carpetas.
- El archivo `Ossium3D.bat` debe ejecutarse desde la carpeta principal del proyecto.
- El primer inicio puede demorar algunos minutos mientras se cargan los componentes del sistema.
