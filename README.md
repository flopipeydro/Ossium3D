# Ossium3D

Bienvenidos al repositorio de nuestro Proyecto Final de Carrera (PFC) en Bioingeniería   
Creamos una **herramienta open-source** que permite diseñar prótesis de cadera personalizadas a partir de imágenes médicas reales, lista para exportar e imprimir en 3D.  
Todo integrado en una sola app. Simple. Flexible. Y pensada para todos.

---

## ¿Qué hace este software?

- Segmenta imágenes médicas (como tomografías).  
- Toma medidas anatómicas del paciente.  
- Modela una prótesis femoral adaptada a ese cuerpo.  
- Exporta el modelo en formato STL para impresión 3D.  

---

## Tecnologías que usamos

- **Python** – Lenguaje base.
- **VTK** – Para visualizar y procesar en 3D.
- **FreeCAD** – Para generar la prótesis con parámetros.
- **PyQt5** – Para la interfaz.
- **STL** – El formato que imprimen todas las impresoras 3D.

---

## ¿Y cómo funciona?

1. Subís una imagen médica (CT de cadera).
2. El sistema la segmenta y reconstruye el hueso.
3. Elegís medidas clave para diseñar tu prótesis.
4. Se genera un modelo 3D personalizado.
5. Exportás el archivo.
   
Para una explicación más detallada, ingresar en Documentos a LUP Ossium3D.
